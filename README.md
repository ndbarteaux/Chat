# Chat
USAGE:
	To comple:
  $ make
	Start server first by running command 
  $ ./chat
	This will give you IP address (IP) and port number (PORT)
	run the command 
  $ ./chat -p <PORT> -s <IP>
	The commands can be in any order, as long as PORT follows -p and IP follows -s
	Type 
  $ ./chat -h  for help
